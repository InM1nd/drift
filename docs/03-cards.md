# 03 · Карты (Протоколы)

## Атомарные эффекты (7 видов)

Каждый Протокол — список из 1–2 атомарных эффектов плюс набор тегов. Статусы ([02 · Боевая петля](02-combat.md)) глобально модифицируют эти атомы, а не прописаны в каждой карте отдельно — отсюда комбинаторная глубина при небольшом пуле карт.

| Эффект | Что делает | Значение |
|---|---|---|
| ДатьУрон | Наносит урон цели | число **или** динамическое (например, ×2 от стека Коррозии на цели) |
| ДатьЩит | Даёт Щит игроку | число **или** динамическое (например, = число сыгранных карт за ход) |
| ДатьЗдоровье | Лечит HP | число |
| ДатьЗаряд | Даёт доп. Заряд в этот ход | число |
| НаложитьСтатус | Накладывает статус на цель | вид статуса + стек |
| ДобратьКарты | Добор карт | число |
| СнизитьСтоимость | Снижает стоимость следующей сыгранной карты | число |

### Динамические значения и Power-триггеры

Часть карт использует не фиксированное число, а значение, привязанное к игровому параметру: текущему Щиту, стеку Коррозии на цели, числу карт, сыгранных за ход. Это финишеры и комбо-payoff карты — именно они создают ощущение билда, поэтому не укладываются в «просто число».

Карты типа **Способность (Power)** не выполняются сразу при розыгрыше — они регистрируют триггер (начало хода, конец хода, розыгрыш N-й карты за ход, получение урона), который сработает позже. В таблицах ниже это видно по формулировке «В начале/конце каждого хода: …» или «Триггер: …».

Точная TS-модель этих типов — в [07 · Архитектура](07-architecture.md), написана согласованно с этим списком карт.

## Архетипы

Четыре направления для компактного прототипа. Большинство карт задевают 1–2 архетипа одновременно, чтобы билды пересекались, а не были изолированы.

- **Коррозия** — накопление стека Коррозии, финишеры «урон = стек».
- **Щит** — генерация Щита, конверсия блока в урон, Стабилизация.
- **Протокол-цепочка** — дешёвые карты, добор, «урон/бонус = карт сыграно».
- **Форсаж** — накопление Форсажа (Сила), редкие, но огромные удары.

## Правило апгрейда

Апгрейд («Протокол+») увеличивает основной числовой эффект примерно на 40–50% (округление в пользу игрока), не меняя Заряд — кроме отдельных карт, задуманных как «дешевеющие». Если у карты есть Exhaust, апгрейд иногда убирает его вместо роста числа. Это не ×2 объём контента по балансу — числа считаются по формуле, а не придумываются для каждой карты заново; ручные исключения делаем только там, где это даёт интересный качественный скачок.

Примеры:
- **Залп → Залп+:** 6 урона → 9 урона (рост числа).
- **Едкий заряд → Едкий заряд+:** 4 урона + 2 Коррозии → 4 урона + 4 Коррозии (акцент на статус — усиливает архетип сильнее, чем прямой урон).
- **Ускоренный протокол → Ускоренный протокол+:** 3 урона, Exhaust → 3 урона, без Exhaust (карта остаётся в колоде на весь бой вместо роста цифр).

Полный список апгрейдов для всех 30 карт — расчёт по этой формуле, не отдельное творческое упражнение; можно сгенерировать таблицей, когда дело дойдёт до данных в коде.
