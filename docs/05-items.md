# 05 · Реликвии и зелья

## Модули — якоря архетипов

Салvажные технологические трофеи. Часть — «якоря»: резко усиливают конкретный архетип и тем самым подсказывают игроку, куда вести колоду. Часть — нейтральные, общее усиление.

| Модуль | Эффект | Архетип-якорь |
|---|---|---|
| Нанитный резервуар | Коррозия на врагах никогда не уменьшается сама по себе (любой источник, включая самоналоженную Коррозию Ядра-Стража) | Коррозия |
| Отражающая обшивка | В конце хода нанести врагу урон, равный текущему Щиту | Щит |
| Взломанный чип приоритета | Первая карта, сыгранная за ход, стоит на 1 Заряд меньше | Протокол-цепочка |
| Боевой рекордер | Форсаж, полученный в бою, не сбрасывается между боями этого захода | Форсаж |

Нейтральные примеры (не привязаны к архетипу): «+1 Заряд каждый 3-й ход», «первая карта в бою в этом заходе бесплатна».

## Инъекторы — тактические разовые

С прицеливанием, дропаются с боёв, инвентарь ограничен (например, 3 слота):

- **Стим форсажа** — временный Форсаж на этот бой.
- **Щитовой инъектор** — мгновенный Щит.
- **Боевой стимулятор** — двойной урон следующей атаки.
- **Медгель** — лечение HP.
- **Реакторный ускоритель** — доп. Заряд на этот ход.
- **ЭМИ-граната** — Помехи по всем врагам.
