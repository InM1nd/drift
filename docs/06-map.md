# 06 · Карта забега

Ветвящийся граф узлов внутри остова «Хорда», путь выбирает игрок; следующий слой виден полностью, дальше — частично.

```
Старт → Отсек → Сигнал/Отсек → Страж → Терминал снабжения → Ремонтный отсек → Ядро-Страж
```

- **Отсек** — обычный бой.
- **Страж** — элитный бой, награда — Модуль.
- **Сигнал** — случайное событие с выбором последствий (найденные данные, аномалия, ловушка).
- **Терминал снабжения** — покупка Протоколов/Модулей/Инъекторов и удаление карты из колоды (важный инструмент курации билда).
- **Ремонтный отсек** — восстановление HP или апгрейд одного Протокола на выбор («Протокол+», см. [03 · Карты](03-cards.md)).
- **Ядро-Страж** — босс акта.

**Риск объёма:** апгрейд карты в Ремонтном отсеке означает, что у каждого Протокола нужна продуманная «+»-версия — см. правило апгрейда в [03 · Карты](03-cards.md), не отдельная задача здесь.

## Реиграбельность

- **Сид захода** — фиксирует граф узлов, дропы, ассортимент терминала для конкретной попытки.
- **Выбор карты после боя** — 4 варианта из пула, можно скипнуть.
- **Архетипы и теги** — комбинаторика из [03 · Карты](03-cards.md) даёт разнообразие даже при одинаковом пуле карт.
- **Уровень угрозы** — модификатор сложности (по образцу Ascension), разблокируется после первой победы над Ядром-Стражем, продлевает интерес без нового контента.
