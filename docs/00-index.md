# ДРЕЙФ (рабочее название) — индекс документации

Пиксельный карточный roguelike-декбилдинг. Веб + PWA, играется на macOS, Windows и телефоне из одного кода.

> **Источник истины:** начиная с этой ревизии, канонической документацией являются файлы в `docs/`. Более ранний HTML-артефакт (`game-design-doc.html`) — устаревший снапшот первой версии плана (Godot-стек, без сеттинга и полного списка карт). Он не обновляется дальше; при расхождениях верить файлам в `docs/`.

## Факты о проекте

| | |
|---|---|
| Стек | Vite + React + TypeScript, PWA |
| Платформы | macOS · Windows · телефон (браузер / «Установить приложение») |
| Ориентация | портрет (основная, и на десктопе тоже) |
| Управление | тап-выбор → тап-цель |
| Масштаб v1 | 1 класс, 1 акт, ~30 карт |
| Источник глубины | теги и архетипы, не объём контента |
