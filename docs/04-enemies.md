# 04 · Враги

Враги — это загадка, которую решают синергии колоды: без них нельзя ни протестировать бой, ни проверить комбо. Для прототипа это не «ИИ», а данные — HP, набор действий и телеграфируемый паттерн намерений (intent), а не спрятанное решение. Игрок всегда видит, что враг сделает в следующий ход, и планирует под это.

Враг описывается так же декларативно, как карта — параллельно `CardData` (см. [07 · Архитектура](07-architecture.md)):

| Поле | Смысл |
|---|---|
| hp | Запас здоровья (диапазон для лёгкого разброса) |
| moveset | Список действий: урон, защита, наложить статус, баф себя |
| pattern | Как выбирается следующий ход: цикл, веса, условие по HP |

## Рядовые враги отсека «Хорда»

**Санитарный дрон** — HP 12–15. Простой агрессор: атакует каждый ход, урон растёт после 3-го хода (перегрев). Паттерн: фиксированный цикл «атака → атака → усиленная атака».

**Турель обшивки** — HP 18–22, стационарная. Чередует накопление щита и залп: «щит → залп → щит → залп». Хорошая цель для карт «урон = стек Коррозии», раз щит долго держится.

**Заражённый образец** — HP 14–18. Каждый ход накладывает Коррозию на игрока вместо прямого урона первые 2 хода, затем переходит в прямую атаку. Учит игрока распознавать, что «тихий» враг не значит безопасный.

## Элита

**Страж-гексапод** — HP 45–55, даёт Модуль (реликвию) при победе. Трёхфазный цикл: «атака + наложить Пробоину → щит на себя → атака по всем картам в руке игрока (штраф за большую руку)». Последнее — мягкий контрпик архетипу Протокол-цепочка, заставляет думать о размере руки.

## Босс акта 1

**Ядро-Страж «Корневой процесс: КОРРОЗИЯ»** — HP ~150. Корневой ИИ остова, заражённый собственным экспериментом.

- **Фаза 1 (HP > 50%):** цикл «атака → накладывает Коррозию на игрока → щит на себя».
- **Фаза 2 (HP ≤ 50%):** паттерн меняется — призывает разовую «Санитарный дрон»-подмогу и накладывает Коррозию по всем целям сразу (то есть себе тоже — можно играть на это картами усиления Коррозии).

Смена фазы телеграфируется заранее (иконка intent меняется на «перегрузка» за ход до порога), чтобы игрок успел подготовиться, а не получил сюрприз.
